# Product Philosophy

Every tool should satisfy at least one of these:

- Solve a real problem.
- Save users time.
- Teach something useful.
- Be enjoyable to use.
- Be easy to share.

Avoid:

- Gimmicks.
- AI for the sake of AI.
- Unnecessary complexity.

## Design Philosophy

The website should feel:

- Lightweight
- Modern
- Professional
- Minimal
- Fast

The tool should always be the focus.

## Growth Philosophy

Priorities:

1. Build useful tools.
2. Improve SEO.
3. Share useful content.
4. Build trust.
5. Grow traffic.
6. Convert visitors into followers.
7. Monetize responsibly.
